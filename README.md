# BackupRidge — Release Archives

Official release archives for [BackupRidge](https://backupridge.com), the WordPress backup & restore plugin.

## Download

Download the latest version from the [Releases](https://github.com/backupridge/backupridge-releases/releases) page.

Each release includes:

- **`backupridge-free-X.Y.Z.zip`** — Free version with full site backup, scheduled automation, one-click restore, and cloud storage support.

## Installation

1. Download the latest `.zip` from [Releases](https://github.com/backupridge/backupridge-releases/releases).
2. In your WordPress dashboard, go to **Plugins → Add New → Upload Plugin**.
3. Choose the downloaded zip file and click **Install Now**.
4. Click **Activate**.

Alternatively, search for **BackupRidge** in **Plugins → Add New** once the plugin is available on [WordPress.org](https://wordpress.org/plugins/).
